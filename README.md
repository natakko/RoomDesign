# Room Design Interior (Unity, Blender, C#)
Дизайн интерьеров и микро-взаимодействий
1. Проектирование жилого пространства с использованием собственных ассетов (мебель, декор).
2. Настройка физики мелких объектов (Rigidbodies) для повышения реализма и интерактивности окружения.
3. Реализация системы освещения с управляемыми источниками света (скрипт выключателя).
